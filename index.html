Previous Projects


Quiz App

This project is a short quiz of three questions where each question is a different format: multiple choice, multiple answer, or true-false. After submitting the answers to all three questions, it reports back to you how many you got correct out of the three.

To Do App

This project allows the user to keep a virtual checklist of what they need to get done. Each item has an empty checkbox and when selected, the box is checked and the item is crossed out. User can also add new items.

Menu App

This project mimics the menu of a hypothetical restaurant, but gets its information from an external source so that every time the restaurant wants to change it, they don't have to go into the code and change it from there.

Exercise App

This project has components such as a timer with a start, stop and reset button and a feature where the user could keep track of their reps by pressing an increment button. Each component is designed for a different type of exercise.
